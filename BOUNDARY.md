# Mac Studio Lab Boundary

This Mac Studio is outside the Kestrel CMMC boundary by default.

Do not store or process:
- CUI
- FCI
- ITAR/export-controlled data
- customer-sensitive data
- proposal-sensitive data
- government-furnished information
- production Kestrel secrets
- production customer credentials
- compliance-controlled source or data

Allowed:
- personal lab code
- public/open-source data
- synthetic data
- generated datasets
- local AI experiments
- Kestrel-adjacent development only with synthetic data and explicit repo rules
